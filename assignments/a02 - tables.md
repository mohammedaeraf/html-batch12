# 📝 Assignment: Book Catalog

**Topic:** HTML Tables
**Duration:** 30 Minutes

## Objective

Create a webpage that displays a **Book Catalog** using an HTML table.

---

## Problem Statement

Create a webpage titled **"Library Book Catalog"**.

Display the following information in a table.

| Book ID | Book Title               | Author           | Category    | Price |
| ------- | ------------------------ | ---------------- | ----------- | ----: |
| B101    | Atomic Habits            | James Clear      | Self Help   |  ₹499 |
| B102    | Clean Code               | Robert C. Martin | Programming |  ₹650 |
| B103    | The Alchemist            | Paulo Coelho     | Fiction     |  ₹399 |
| B104    | The Pragmatic Programmer | Andrew Hunt      | Programming |  ₹799 |
| B105    | Rich Dad Poor Dad        | Robert Kiyosaki  | Finance     |  ₹450 |

---

## Requirements

1. Create a webpage with the title **Library Book Catalog**.
2. Display a heading **Library Book Catalog** at the top of the page.
3. Create a table with **5 columns**.
4. Use the following HTML elements:

   * `<table>`
   * `<thead>`
   * `<tbody>`
   * `<tr>`
   * `<th>`
   * `<td>`
5. Add a border to the table.
6. Collapse the table borders using `border-collapse`.
7. Add **10px padding** to all table cells.
8. Center the table on the page.
9. Center-align all table data.
10. Style the header with:

    * Blue background
    * White text
11. Apply alternate row colors using `tr:nth-child(even)`.
12. Add a hover effect that changes the background color of the row.
