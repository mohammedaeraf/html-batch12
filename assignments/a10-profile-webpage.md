# Assignment – HTML Internal Links

## Title: My Personal Website

### Objective

Create a simple **single-page website** using **HTML Internal Links** and **Bootstrap**.

In this assignment, you will practice:

* Creating internal links
* Using the `id` attribute
* Navigating between different sections of the same webpage
* Applying Bootstrap utility classes for styling
* Creating a simple navigation menu

---

# Requirements

## 1. Add Bootstrap

Include the Bootstrap 5 CDN in your HTML file.

---

## 2. Create a Page Heading

Create the following heading:

**My Personal Website**

Apply suitable Bootstrap classes to make it:

* Center aligned
* Blue (`text-primary`)
* Large (`display-4`)
* Bold (`fw-bold`)
* Margin on top and bottom (`my-5`)

---

## 3. Create a Navigation Menu

Create a navigation menu using Bootstrap buttons.

The menu should contain the following links:

* Home
* About Me
* Skills
* Contact

Each button should navigate to its corresponding section using **internal links**.

---

## 4. Create Four Sections

Create the following sections on the same webpage.

### Home

Include:

* Heading
* A short welcome message

Use Bootstrap classes:

* `bg-primary-subtle`
* `p-4`
* `rounded`
* `shadow`
* `mb-5`

---

### About Me

Include:

* Heading
* A short paragraph introducing yourself.

Use:

* `bg-success-subtle`
* `p-4`
* `rounded`
* `shadow`
* `mb-5`

---

### Skills

Include a short paragraph mentioning some of your skills.

Example:

* HTML
* CSS
* Bootstrap
* JavaScript

Use:

* `bg-warning-subtle`
* `p-4`
* `rounded`
* `shadow`
* `mb-5`

---

### Contact

Include:

* Email Address
* Phone Number

Use:

* `bg-danger-subtle`
* `p-4`
* `rounded`
* `shadow`
* `mb-5`

---

## 5. Add a Back to Top Button

At the bottom of the page, create a **Back to Top** button.

The button should navigate to the **Home** section using an internal link.

Use:

* `btn`
* `btn-dark`

---

## Learning Outcome

By completing this assignment, you will learn how to create internal links using the `id` attribute and `href="#id"` while building a clean, well-organized single-page website. You will also strengthen your understanding of Bootstrap buttons, spacing, colors, and utility classes to create an attractive and user-friendly navigation experience.
