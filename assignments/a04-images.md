# Assignment – Bootstrap Images

## Theme: Healthy Fruits Gallery

### Objective

Create a simple Bootstrap webpage showcasing healthy fruits with responsive images and Bootstrap image classes.

---

## Requirements

### 1. Add Bootstrap

Include the Bootstrap 5 CDN in your HTML file.

---

### 2. Create a Heading

Create the following heading:

**Healthy Fruits Gallery**

Apply these Bootstrap classes:

* `text-danger`
* `text-center`
* `display-4`
* `fw-bold`
* `my-5`

---

### 3. Create Three Fruit Sections

Each section should contain:

* Fruit Name (Heading)
* Fruit Image
* Short description (3–4 lines)

Use the following fruits:

* 🍎 Apple
* 🍌 Banana
* 🥭 Mango

---

### 4. Apple Section

Apply these classes to the image:

* `img-fluid`
* `rounded`
* `shadow`

The container should have:

* `border`
* `rounded`
* `p-4`
* `mb-5`

---

### 5. Banana Section

Apply these classes to the image:

* `img-thumbnail`

The container should have:

* `border`
* `border-warning`
* `rounded-4`
* `p-4`
* `mb-5`

---

### 6. Mango Section

Apply these classes to the image:

* `img-fluid`
* `border`
* `border-3`
* `border-success`
* `rounded-5`
* `shadow-lg`

The container should have:

* `border`
* `border-success`
* `rounded-4`
* `p-4`

---
