# Assignment – Bootstrap Cards

## Title: Electronic Store

### Objective

Create a modern and attractive **Electronic Store** webpage using **Bootstrap Cards**.

In this assignment, you will practice:

- Creating Bootstrap Cards
- Adding images to cards
- Using card titles and descriptions
- Adding Bootstrap buttons
- Applying Bootstrap utility classes for spacing and styling

---

# Requirements

## 1. Add Bootstrap

Include the Bootstrap 5 CDN in your HTML file.

---

## 2. Create a Page Heading

Create the following heading:

**Electronic Store**

Apply suitable Bootstrap classes to make it:

- Center aligned
- Blue (`text-primary`)
- Large (`display-4`)
- Bold (`fw-bold`)
- Margin on top and bottom (`my-5`)

---

## 3. Create Three Product Cards

Create **three Bootstrap cards**, one below the other, for the following products:

- 💻 Laptop
- 📱 Smartphone
- 🎧 Wireless Headphones

Each card should include:

- Product Image
- Product Name
- Short Description (3–4 lines)
- Product Price
- **Buy Now** button
- **View Details** button

---

## 4. Sample Product Details

You may use the following details or create your own.

### 💻 Laptop

A high-performance laptop suitable for programming, office work, gaming, and multimedia tasks. It features a fast processor, ample storage, and a long-lasting battery.

## 5. Apply Bootstrap Classes

Use the following Bootstrap classes wherever appropriate.

### Card Classes

- `card`
- `card-body`
- `card-title`
- `card-text`
- `card-img-top`

### Buttons

- `btn`
- `btn-primary`
- `btn-outline-primary`

### Utility Classes

- `shadow`
- `rounded`
- `mb-4`
- `me-2`
- `container`
- `my-5`

# Bonus Challenge ⭐

Complete **any two** of the following:

1. Add a **fourth product card** (Smart Watch, Tablet, Camera, or Gaming Mouse).

2. Use different Bootstrap button colors for each product (for example, `btn-success`, `btn-warning`, `btn-danger`).

3. Display the product price in **green** using the `text-success` class.

4. Add a **"Special Offer"** badge or label above one of the product cards using a Bootstrap Alert.

---

# Submission Guidelines

- Save the file as **`electronic-store.html`**.
- Create an **`images`** folder and store all product images inside it.
- Ensure all images are displayed correctly.
- Use **Bootstrap classes only** for styling. Do **not** use custom CSS.
- Test the webpage in a browser before submitting.

---

## Learning Outcome

By completing this assignment, you will learn how to build visually appealing **Bootstrap Cards** that include images, text, buttons, and utility classes. This assignment reinforces the use of responsive card layouts commonly found in e-commerce websites and online product catalogs.
