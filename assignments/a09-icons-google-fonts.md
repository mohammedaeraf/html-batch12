# Assignment – Google Fonts & Bootstrap Icons

## Title: Enhance the "Explore Amazing Destinations" Webpage

### Objective

Enhance your existing **Explore Amazing Destinations** webpage by adding **Google Fonts** and **Bootstrap Icons** to create a more modern and professional user interface.

In this assignment, you will practice:

* Adding Google Fonts to a webpage
* Applying a custom font using CSS
* Adding Bootstrap Icons
* Using icons in headings, alerts, cards, prices, and buttons
* Improving the visual appearance of a Bootstrap webpage

---

# Requirements

## 1. Add Google Font

Visit **Google Fonts** and select the **Poppins** font.

Add the Google Fonts `<link>` tags inside the `<head>` section.

Apply the **Poppins** font to the entire webpage using CSS.

---

## 2. Add Bootstrap Icons

Include the Bootstrap Icons CDN inside the `<head>` section.

---

## 3. Improve the Main Heading

Modify the page heading:

**Explore Amazing Destinations**

Add an **Airplane** icon before the heading.

---

## 4. Improve the Alert

Update the existing offer alert.

Add a **Megaphone** icon before the offer message.

Example message:

**Special Offer! Get 15% OFF on all holiday packages this month!**

---

## 5. Add Icons to Destination Names

Add a suitable icon before each destination name.

Suggested icons:

* 📍 Goa → Location Icon
* 🌳 Munnar → Tree Icon
* ☀ Ooty → Sun or Cloud-Sun Icon

---

## 6. Improve the Price Display

Display the package price in green.

Add a **Rupee** icon before the price.

Example:

**Starting from ₹14,999**

---

## 7. Improve the Buttons

Modify the buttons as follows:

### Explore Button

* Add a **Compass** icon.

### Book Now Button

* Add a **Calendar Check** icon.

---

## 8. Keep the Existing Responsive Layout

Ensure that the webpage continues to use:

* `container`
* `row`
* `col-md-4`
* `g-4`
* `card`
* `shadow`
* `rounded`
* `h-100`

The layout should remain responsive on mobile, tablet, and desktop devices.
