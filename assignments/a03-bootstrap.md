# Assignment: Create a Travel Destination Page

### Objective

Create a simple Bootstrap webpage highlighting two tourist destinations.

### Requirements

1. Add Bootstrap CSS using CDN.

2. Create a heading

* Text: **Beautiful Tourist Destinations**
* Blue color
* Center aligned
* Display-3
* Bold
* Margin on all sides

3. Create the first destination section

* Title: **Goa**
* Paragraph describing Goa (4–5 lines)
* Light success background
* Success colored border (2px)
* Padding
* Margin
* Rounded corners
* Shadow

4. Create the second destination section

* Title: **Munnar**
* Paragraph describing Munnar (4–5 lines)
* Light info background
* Info colored border (3px)
* More padding than first section
* Margin
* Rounded-pill or Rounded-4
* Large shadow

5. Use Bootstrap classes only for styling.
