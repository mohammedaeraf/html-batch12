# 📝 Assignment: HTML Lists

**Topic:** HTML Lists (`<ul>` and `<ol>`)
**Duration:** 20 Minutes

## Objective

Create a webpage that demonstrates the use of **Ordered Lists** and **Unordered Lists**.

---

## Problem Statement

Create a webpage with the title **"My Favorite Sandwich Recipe"**.

The page should contain the following sections:

### Ingredients (Unordered List)

* Bread
* Butter
* Tomato
* Cucumber
* Cheese
* Salt
* Pepper

---

### Preparation Steps (Ordered List)

1. Take two slices of bread.
2. Spread butter on both slices.
3. Add cucumber and tomato slices.
4. Sprinkle salt and pepper.
5. Place the cheese on top.
6. Cover with the second slice of bread.
7. Cut the sandwich into two pieces.
8. Serve fresh.

---

## Requirements

1. Create an HTML page with the title **My Favorite Sandwich Recipe**.
2. Display the main heading:

   ```
   My Favorite Sandwich Recipe
   ```
3. Add a subheading **Ingredients** and display the items using an **Unordered List (`<ul>`)**.
4. Add another subheading **Preparation Steps** and display the steps using an **Ordered List (`<ol>`)**.
5. Use appropriate heading tags (`<h1>` and `<h2>`).

---

## Expected Output

```text

Ingredients
```
