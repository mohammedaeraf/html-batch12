# Assignment – Bootstrap Alerts & List Groups

## Title: Programming Courses Portal

### Objective

Create a simple webpage for a **Programming Courses Portal** using Bootstrap **Alerts** and **List Groups**.

In this assignment, you will practice:

- Bootstrap Alerts
- Bootstrap List Groups
- Bootstrap Typography
- Bootstrap Utility Classes

---

# Requirements

## 1. Add Bootstrap

Include the Bootstrap 5 CDN in your HTML file.

---

## 2. Create a Page Heading

Create the following heading:

**Programming Courses Portal**

Apply suitable Bootstrap classes to make it:

- Center aligned
- Blue (`text-primary`)
- Large (`display-4`)
- Bold (`fw-bold`)
- Margin on top and bottom (`my-5`)

---

## 3. Create Two Alerts

### Alert 1 – Welcome Message

Create a **Success Alert** displaying the following message:

> ✅ Welcome to the Programming Courses Portal!

---

### Alert 2 – Assignment Reminder

Create a **Warning Alert** displaying the following message:

> ⚠ Submit your assignment before Friday to avoid late submission.

---

## 4. Create a List Group

Below the alerts, add the heading:

**Available Courses**

Create a Bootstrap **List Group** with the following items:

- Available Courses _(Active Item)_
- HTML
- CSS
- Bootstrap
- JavaScript
- React
- PHP

Use the **active** class for the first item.

---

## 5. Apply Bootstrap Utility Classes

Use suitable Bootstrap classes to improve the appearance of your webpage.

Suggested classes:

### Typography

- `text-center`
- `text-primary`
- `display-4`
- `fw-bold`

### Alerts

- `alert`
- `alert-success`
- `alert-warning`

### List Groups

- `list-group`
- `list-group-item`
- `active`

### Spacing

- `container`
- `my-5`
- `mb-4`
- `mt-4`

# Submission Guidelines

- Save the file as **`courses.html`**.
- Test the webpage in a browser before submitting.
- Use **Bootstrap classes only** for styling. Do **not** use custom CSS.

## Learning Outcome

By completing this assignment, you will learn how to use Bootstrap **Alerts**, **List Groups**, **Typography**, and **Utility Classes** to create a clean, organized, and user-friendly webpage.
