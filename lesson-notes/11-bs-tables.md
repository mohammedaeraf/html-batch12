
# Practice Exercise

Create a webpage titled **"Employee Salary Report"**.

The page should contain:

* A centered heading: **Employee Salary Report**
* A responsive Bootstrap table with the following columns:

  * Employee ID
  * Name
  * Department
  * Salary
  * Experience
* Add at least **6 employee records**.
* Apply the following Bootstrap classes:

  * `table`
  * `table-striped`
  * `table-bordered`
  * `table-hover`
* Use `table-dark` for the table header.
* Wrap the table inside a `table-responsive` container.

This exercise will help you learn how to present tabular data professionally using Bootstrap Tables.
